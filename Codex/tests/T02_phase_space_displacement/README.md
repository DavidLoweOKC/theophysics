# Test 2: Phase Space Displacement

## What This Tests
Whether constrained behavioral displacement produces structured compensation rather than random drift.

## Why It Matters
If compensation is random instead of conserved, the conservation-style moral displacement claim weakens.

## Planned Implementation
Monte Carlo beta-decay style simulation with constrained channels and compensation tracking.

## Pass Criteria
When one channel is constrained, the displaced term rises in a structured, conservation-like way.

## Fail Criteria
Compensation is absent, unstable, or random.

## Kill Condition
If displacement is not structured, the moral conservation framing needs revision.

# Test 6: Network Fragmentation

## What This Tests
Whether a bonded network fragments without external sustaining input and remains coherent with it.

## Why It Matters
This is the open-system requirement expressed as a network dynamics test.

## Canonical Inputs Used
- `canonical/CODEX_TEST_BRIEFING.md`
- `canonical/MASTER_TEST_STACK.md`
- `canonical/FRUITS_OF_SPIRIT.md`

## Planned Implementation
Weighted graph simulation with decaying bonds and periodic restorative input.

## Pass Criteria
The unsupported network fragments while the supported network remains cohesive.

## Fail Criteria
The unsupported network stays stable or the supported network gains no meaningful advantage.

## Kill Condition
If the unsupported network holds together indefinitely, the open-system claim weakens.

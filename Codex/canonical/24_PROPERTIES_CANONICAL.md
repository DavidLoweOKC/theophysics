## How to Read This Table

**Positive Extreme:** Max the property — run it to the edge. Does reality hold?
**Negative Extreme:** Negate it, max that out. What breaks?
**Math Status:**
- **CORE** = math dies without it (6 irreducibles)
- **DERIVED** = real but reduces to a core property
- **RELATIONAL** = describes math's relationship to other things
**Kill Condition:** What would falsify the match?

## The 6 Irreducible Core

Without these, math doesn't weaken — it dies.

| # | Property | Death Clause |
|---|----------|-------------|
| 4 | Simple | No primitives = no starting point = no proof begins |
| 5 | Consistent | Contradiction = explosion = everything provable = nothing informative |
| 12 | True | No truth values = no difference between proof and error |
| 13 | Rational | Inference fails = nothing follows from anything |
| 23 | Generative | Axioms inert = no theorems = no math |
| 24 | Judging | Can't sort = can't verify = function gone |

**Read as one sentence:** A system that bears truth, maintains non-contradiction, evaluates its own statements, operates by logical inference, starts from irreducible primitives, and produces results from those primitives.

That is Logos. Not analogy — the six are what Logos looks like stripped to bones.
